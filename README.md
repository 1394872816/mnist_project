# MNIST 手写数字识别项目

基于 PyTorch 的 CNN 手写数字识别项目，支持实时 GPU 监控和训练可视化。

## 项目特点

- **CNN模型**: 自定义设计的卷积神经网络，用于数字分类
- **实时监控**: 训练过程中实时显示 GPU/CPU 使用情况
- **进度条显示**: 使用 tqdm 显示训练进度
- **可视化功能**: 
  - 训练/测试损失曲线
  - 测试准确率曲线
  - 预测结果网格图（绿色=正确，红色=错误）
  - 混淆矩阵
- **模块化设计**: 代码结构清晰，易于维护和扩展

## 环境要求

- Python 3.8+
- PyTorch 1.9+
- CUDA 11.0+（可选，用于GPU加速）
